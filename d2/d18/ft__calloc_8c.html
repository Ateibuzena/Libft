Asigna memoria y la inicializa a cero.

Esta función reserva un bloque de memoria del tamaño especificado y lo inicializa a cero. La memoria reservada se libera mediante free.

Parameters
countEl número de elementos a reservar.
sizeEl tamaño de cada elemento.
Returns
Un puntero al bloque de memoria reservado e inicializado a cero, o NULL si la asignación falla.
