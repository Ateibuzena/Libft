int ft_isprint (int c)
 

Detailed Description

Verifica si un carácter es imprimible.

Comprueba si el carácter proporcionado como argumento es un carácter imprimible, incluidos los caracteres visibles y el espacio (códigos ASCII del 32 al 126).

Parameters
cEl carácter a comprobar, representado como un entero.
Returns
  • 1 si el carácter es imprimible.
  • 0 en caso contrario.
Note
La función también verifica si el carácter pertenece al conjunto ASCII utilizando ft_isascii.
