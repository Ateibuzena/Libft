int ft_isdigit (int c)
 

Detailed Description

Verifica si un carácter es un dígito decimal.

Comprueba si el carácter proporcionado como argumento es un dígito decimal (0-9) comparando su valor ASCII.

Parameters
cEl carácter a comprobar, representado como un entero.
Returns
  • 1 si el carácter es un dígito decimal.
  • 0 en caso contrario.
