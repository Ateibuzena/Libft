void ft_putchar_fd (char c, int fd)
 

Detailed Description

Escribe un carácter en un archivo descriptor.

Esta función escribe un solo carácter c en el archivo o descriptor de archivo identificado por el descriptor fd.

Parameters
cEl carácter a escribir.
fdEl descriptor de archivo en el que se escribe el carácter.
Note
La función utiliza la llamada al sistema write para escribir el carácter en el archivo.
